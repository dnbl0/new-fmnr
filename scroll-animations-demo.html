Patterns

Each animation pattern used across the FMNR website, documented in isolation. Scroll down to trigger each demo as it enters the viewport.

Pattern 01

Fade-Up Reveal

Elements fade in and slide up as they enter the viewport. Used for headings, paragraphs, and section content throughout the site. Triggered once on scroll.

Pattern 03

Parallax Scroll

Background images move at a slower rate than the page scroll, creating depth. Used on hero sections and full-bleed image bands. The image shifts vertically as you scroll past.

Pattern 04

Scale-In (Bounce)

Elements scale from small to full size with a slight overshoot/bounce. Used for stat numbers, icons, and emphasis moments. Draws attention with a springy entrance.

Pattern 05

Animated Count-Up

Numbers count from zero to their target value when scrolled into view. Used in stat strips, impact numbers, and data visualisation sections. Creates a sense of scale and achievement.

Pattern 06

Slide-In from Sides

Elements slide in from the left or right. Used for two-column layouts, split content sections, and bio panels. Creates a sense of two halves coming together.

Bar Chart Fill

Pattern 08

Ken Burns (Slow Zoom)

A subtle continuous zoom/pan on hero images that adds life without requiring interaction. Used on page hero sections. The image gently scales and translates over time using CSS keyframes.

Summary

When to use each pattern

Fade-up — Default for all content entering the viewport.
Stagger — Grids of 3+ items (cards, pillars, awards).
Parallax — Hero images and full-bleed photo bands.
Scale-in — Key stat numbers or icons that need emphasis.
Count-up — Numeric data (impact stats, beneficiary counts).
Slide-in — Two-column split layouts (bio, feature sections).
Bar chart — Data comparison (adoption rates, percentages).
Ken Burns — Hero images to add subtle life on load.
