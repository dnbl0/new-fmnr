WordPress Implementation

How to replicate this component using WordPress Gutenberg plugins.

Recommended Plugin

CoBlocks (by GoDaddy) — native Gutenberg block plugin with a Counter block that supports animated count-up, prefixes/suffixes (%, M+), and customisable typography.

Alternative: Flavor Counter Block — lightweight, single-purpose animated counter for Gutenberg.

Block Structure

core/columns (4 columns)
→ coblocks/counter per column
→ core/paragraph for label

Styling (Additional CSS)

.stat-strip { background: var(--wp--preset--color--contrast); }
.stat-strip .coblocks-counter__number {
  font-family: var(--font-heading);
  color: var(--color-teal);
  font-size: clamp(52px, 5.5vw, 80px);
}
.stat-strip .wp-block-column {
  border-right: 1px solid rgba(255,255,255,0.2);
  text-align: center;
  padding: 52px 36px;
}

Scroll Trigger

Wrap the Columns block in Animatikon Motion FX with Opacity (start 0 → end 1) to trigger the entrance on scroll. The CoBlocks counter auto-animates its numbers when visible in viewport.
